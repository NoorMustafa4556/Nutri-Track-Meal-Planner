
# 🥗 Nutri Track – Smart Meal Planner App (Flutter)

**Nutri Track** is a powerful and elegant Flutter application designed to simplify healthy eating. It helps users discover delicious recipes, plan their weekly meals with AI-like smart filtering, and provides an immersive cooking experience with step-by-step guidance.

Whether you are looking for a **Keto, Vegan, or High-Protein** diet, Nutri Track adapts to your preferences and allergies to ensure every meal is perfect for you.

---

## 🚀 Key Features
- **Smart Recipe Discovery** – Explore a wide range of recipes with calories, macros, and ratings.
- **AI-Powered Filtering** – Automatically filter meals based on your **Diet Type** (Keto, Vegan, etc.) and **Allergies** (Nuts, Dairy, etc.).
- **Weekly Meal Planner** – Schedule your meals for the entire week and see your nutritional summary.
- **Immersive Cook Mode** – Step-by-step cooking instructions with a built-in progress tracker and adjustable timer.
- **Favorites & Syncing** – Save your favorite meals and sync your data with **Firebase Cloud Firestore**.
- **Modern UI/UX** – Premium dark mode support, fluid animations, and beautiful typography (Google Fonts).
- **Personalized Profile** – Set your goals and preferences to get a truly tailored experience.

---

### 📖 Immersive Cooking Experience
- **Step-by-Step UI** → Stay focused on one task at a time.
- **Live Progress Bar** → See how much of the recipe is remaining.
- **Smart Timer** → Set and manage cooking timers directly within the app.

### 🕌 Smart Planning & Nutrition
- **Auto-Generate Plan** → Get a smart weekly plan in one tap.
- **Nutrition Tracking** → Monitor your weekly calorie intake automatically.
- **Recipe Picker** → Easily search and swap meals in your schedule.

### 🎨 Premium Design
- **Dual Theme Support** → Beautiful Light and Dark modes.
- **Fluid Transitions** → Smooth animations powered by Flutter's animation engine.
- **Clean Typography** → Uses Poppins and other premium Google Fonts.

## 🚀 Getting Started

### Prerequisites
- Flutter SDK → [flutter.dev](https://flutter.dev)
- Android Studio or VS Code
- Firebase Project Setup

### Installation
**1. Clone the repository:**
```bash
git clone https://github.com/NoorMustafa4556/Nutri-Track-Meal-Planner.git
```

**2. Install dependencies:**
```bash
cd meal_planner_app
```
```bash
flutter pub get
```

**3. Set up Firebase:**
- Add your `google-services.json` (Android) and `GoogleService-Info.plist` (iOS).
- Enable Email/Password Auth & Firestore in Firebase Console.

**4. Run the app:**
```bash
flutter run
```
